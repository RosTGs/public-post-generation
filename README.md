# public-post-generation

Локальный прототип веб-приложения для генерации постов и картинок.

## Быстрый старт

```bash
python3 -m http.server 8000
```

Откройте `http://localhost:8000` в браузере.

## Возможности

- GPT чат для генерации множества постов за раз.
- Библиотека постов с избранным, редактированием и выбором для публикации.
- Имитация генерации изображений Gemini с привязкой к каждому посту.
- Настройки ботов и каналов, выбор для публикации избранных постов.

> Примечание: все данные сохраняются в `localStorage` браузера.
